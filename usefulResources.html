Resource List
The Skills Network

The skills network can help with a number of different qualifications. I took the one specific to software development, which may or may not be available at this time. It has been abundantly helpful for getting started.

LINK: https://theskillsnetwork.com
W3 Schools

W3 Schools can potentially be used to learn from scratch, but may be a bit opaque without someone to help. I still find it extremely useful, both as a reference and to try and find new functions that are useful for whatever code I'm working on.'

LINK: https://www.w3schools.com/python/
Programiz/Visual Studio Code

I've put these two together, because I often use them in tandem. Programiz has an online python compiler, which is useful for quick testing and experimenting. It, however, does not have most libraries available, nor can you save code easily. This is where Visual Studio Code excells, as it's possible to download any library and it of course actually saves files permenantly.

Both of these can be used for free. There's no need to feel compelled to pay for them unless you have a good reason.

LINK: https://www.programiz.com/python-programming/online-compiler/ LINK: https://visualstudio.microsoft.com/downloads/
